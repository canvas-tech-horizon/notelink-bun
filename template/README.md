# {{project-name}}

A NoteLink API project built with Bun and ElysiaJS.

## Getting Started

Install dependencies:

```bash
bun install
```

Run the development server:

```bash
bun dev
```

## Project Structure

```
src/
├── index.ts          # Main application entry point
├── routes/           # API routes
├── handlers/         # Request handlers
└── config/           # Configuration files
```
